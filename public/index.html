We must build a **Backend API** to act as a secure middleman. The best way to do this is to use **Node.js** with the **Express** framework, as it's fast and uses JavaScript, which you'll also use in the frontend. Here is the plan to make your static site dynamic. #### ⚙️ Step-by-Step Execution (Backend) 1. **Modify Your Folder Structure:** Go to your `DiagnosticWebsite` folder. Create a new sub-folder inside it named `public`. Now, move your four HTML files (`index.html`, `patients.html`, etc.) *inside* this new `public` folder. Your structure should look like this: ``` /DiagnosticWebsite /public index.html patients.html tests.html staff.html ``` 2. **Open a Terminal** in the `DiagnosticWebsite` folder (the main one, not `public`). 3. **Initialize Node.js:** Run this command. It creates a `package.json` file to track your packages. ```bash npm init -y ``` 4. **Install Packages:** We need two packages: `express` (the server) and `mysql2` (the modern MySQL driver). ```bash npm install express mysql2 ``` 5. **Create the Server File:** In the main `DiagnosticWebsite` folder (next to `package.json`), create a new file named `server.js`. 6. **Copy this code into `server.js`:** This code does two things: * Serves your static HTML files from the `public` folder. * Creates API "endpoints" (like `/api/patients`) that query your `lifeline_db` database. --- #### Code for `server.js` ```javascript:Backend Server:server.js const express = require('express'); const mysql = require('mysql2'); const path = require('path'); const app = express(); const port = 3001; // We'll run our server on this port // --- 1. DATABASE CONNECTION --- // Create a "pool" of connections to your MySQL database // !! Replace with your own MySQL password !! const db = mysql.createPool({ host: 'localhost', user: 'root', // Or your MySQL username password: 'YOUR_MYSQL_PASSWORD_HERE', // !! IMPORTANT: CHANGE THIS database: 'lifeline_db' }).promise(); // Use .promise() for modern async/await syntax // --- 2. SERVE STATIC FILES --- // This tells Express to serve your HTML, CSS, and JS files // from the 'public' folder. app.use(express.static(path.join(__dirname, 'public'))); // --- 3. CREATE API ENDPOINTS --- // Example: API to get all patients app.get('/api/patients', async (req, res) => { try { const [rows] = await db.query('SELECT * FROM Patient'); res.json(rows); // Send the data back as JSON } catch (err) { console.error('Error fetching patients:', err); res.status(500).json({ error: 'Database error' }); } }); // Example: API to get all scheduled tests app.get('/api/tests', async (req, res) => { try { // Your query might be more complex, but this is a start const [rows] = await db.query('SELECT * FROM Test'); res.json(rows); } catch (err) { console.error('Error fetching tests:', err); res.status(500).json({ error: 'Database error' }); } }); // TODO: Create APIs for staff (doctors, reviewers) // app.get('/api/doctors', async (req, res) => { ... }); // app.get('/api/reviewers', async (req, res) => { ... }); // --- 4. START THE SERVER --- app.listen(port, () => { console.log(`Server running at http://localhost:${port}`); console.log('Serving static files from:', path.join(__dirname, 'public')); }); ```eof --- #### ⚙️ How to Run Your *Full* Application Now that you have the server, you will **no longer double-click the HTML file.** 1. Make sure your MySQL server is running (the one with `lifeline_db`). 2. Open your terminal in the `DiagnosticWebsite` folder. 3. Run your server: ```bash node server.js ``` 4. You will see `Server running at http://localhost:3001`. 5. Open **`http://localhost:3001`** in your web browser. This will load your `index.html` file. Now, when you click "Patients," it will load `patients.html`, but this time it's being served by your backend. --- ### Part 3: Connecting Frontend to Backend One last step.
